# PromptForge

Advanced AI prompt builder — chatbot & image generation.

**Live →** [florianm974.github.io/promptforge](https://florianm974.github.io/prompt-forge)

## Features

- **2 modes** — Chatbot (ChatGPT, Claude, Mistral...) & Image generation (Midjourney, Stable Diffusion...)
- **Real-time prompt builder** — prompt updates as you fill in the fields
- **Templates** — ready-to-use presets for common use cases
- **Tag inputs** — add constraints, examples, quality tags, negative prompts
- **History** — saved locally in the browser (localStorage), up to 20 entries
- **Copy in one click**

## Stack

- HTML / CSS / JS — no framework, no dependencies
- Hosted on GitHub Pages
